Oregon Trail

Created a game called Oregon Trail with a partner. Implemented with classes and principles of object-oriented programming in C++. The user can face a series of obstacles and can choose from a menu. Their progress is tracked until they get to the end of the game or die.

Mini-Git

Implemented a version control system similar to gitHub using data structures such as linked lists and pointers. The user can either choose to add, commit or delete and the linked list holds all the commits they have made.
